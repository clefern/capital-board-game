# Melhorias Futuras - Jogo Capital

## Polimento Visual e UX
1. **Animações de transações** — feedback visual quando dinheiro muda de mãos (números flutuantes, efeitos de partículas)
2. **Indicadores visuais de bônus** — mostrar no tabuleiro quais negócios estão recebendo bônus de marca/região/vizinhança
3. **Histórico de jogadas** — log scrollável com replay das últimas ações
4. **Notificações toast** — alertas mais visíveis para eventos importantes (falência iminente, carta de vitória disponível)

## Jogabilidade
5. **Mais minigames** — atualmente são 4, poderia ter 6-8 para mais variedade
6. **Trocas multi-carta** — atualmente só troca 1:1, poderia permitir 2:1 ou incluir dinheiro na troca
7. **Leilões** — quando um jogador não pode pagar aluguel, leiloar um negócio em vez de vender automático
8. **Eventos aleatórios** — crise econômica, boom imobiliário, etc. que afetam todos

## Técnico
9. **Service Worker** — falta o SW para funcionar offline como PWA real
10. **Responsividade mobile** — layout é só landscape, poderia adaptar para portrait em celular
11. **Atalhos de teclado** — navegar ações com teclas (1-5, Enter, Esc)
12. **Estatísticas persistentes** — histórico de partidas, recordes, win rate por jogador

## Conteúdo
13. **Mais cartas** — expandir de 23 para ~35 tipos para mais variedade estratégica
14. **Modos de jogo** — modo rápido (menos dinheiro para vitória), modo longo, modo cooperativo
